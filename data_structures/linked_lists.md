# Linked List

- Nodes with (value, next).
- No random access → O(n).
- Insert/delete at head → O(1).
- Insert/delete at tail → O(1) with tail pointer.

| Operation         | Time Complexity |
|-------------------|-----------------|
| Push front/back   | O(1)            |
| Pop front         | O(1)            |
| Pop back (singly) | O(n)            |
| Search by value   | O(n)            |
