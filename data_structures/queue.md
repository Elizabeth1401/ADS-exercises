# Queue (FIFO)

- First In, First Out.
- Operations: enqueue(x), dequeue(), front(), is_empty().
- Implement via deque (best), circular array, or linked list.

| Operation | Time Complexity |
|-----------|-----------------|
| enqueue   | O(1)            |
| dequeue   | O(1)            |
| front     | O(1)            |
